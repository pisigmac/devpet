FAQ

Questions? Answered.

How does DevPet track my coding? ▼
The VS Code extension listens for file saves, git commits, and language switches. It batches keystrokes every 50 strokes. All data is sent securely to your Supabase project — we never see your code.
Is my code safe? ▼
Absolutely. We only track metadata: event type, programming language, and file extension. We never read, store, or transmit your actual code. Everything is end-to-end encrypted via Supabase.
Can I use DevPet without VS Code? ▼
Yes! The web dashboard and mobile app work independently. You can manually log sessions or sync GitHub commits. The VS Code extension just makes it automatic.
What payment methods are supported? ▼
We support Stripe for global payments (cards, PayPal, Apple Pay) and Razorpay for India (UPI, NetBanking, cards, wallets). You can switch providers anytime in settings.
Can I self-host DevPet? ▼
Yes! The entire stack is open-source. Deploy your own Supabase backend, host the dashboard on Cloudflare Pages, and sideload the VS Code extension. Full instructions in our GitHub repo.
