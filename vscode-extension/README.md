# DevPet VS Code Extension

## Setup
1. Open VS Code settings (Cmd/Ctrl + ,)
2. Search for "DevPet"
3. Enter your Supabase URL and Anon Key
4. Reload window

## Features
- Real-time coding habit tracking
- Pet evolution based on commits, saves, language diversity
- Status bar mood indicator
- Dashboard panel with full stats
- Git commit detection

## Development
```bash
npm install
npm run compile
F5 to launch Extension Development Host
```

## Packaging
```bash
vsce package
vsce publish
```
